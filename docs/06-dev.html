6.3 从曲线提取"特征"（给模型吃的数据）

特征名含义典型合格区间（无铅）
peak_temp峰值温度235~250 ℃
TAL液相线(217℃)以上时间45~90 秒
ramp_up升温斜率1~3 ℃/秒
ramp_down降温斜率-1 ~ -4 ℃/秒
delta_T板面最大温差< 15 ℃
soak_temp恒温区平台温度150~200 ℃

6.4 三个核心模型

模型1：工艺代理模型（设定值 → 预测曲线）

输入 zone_temp、chain_speed、风量 → 输出预测实际曲线。用途：还没真过炉就能猜曲线。方法：物理热模型+数据校正 / 神经网络。

模型2：质量预测模型（曲线+板+锡膏 → 缺陷概率）

输入 6 个特征 + 板厚/铜面积/BGA有无/锡膏型号 → 输出虚焊/桥连/立碑/空洞概率。方法：XGBoost / 随机森林（小样本友好、可解释）。这是替代老师傅"预判风险"的核心。

模型3：参数推荐引擎（新板 → 最优设定）

输入 新产品BOM+锡膏+工艺窗口约束 → 输出推荐 zone_temp/chain_speed。方法：贝叶斯优化 / 遗传算法。输出同时给"风险提示"和"为什么这么设"（可解释）。

6.6 知识库（把老师傅经验存下来）

访谈老师傅，整理成 "场景→做法→原因" 结构化条目，例如：

▸ 场景：板子有大面积接地铜箔（热容大）
▸ 做法：回流区温度上调 5~8℃，链速降 5 cm/min
▸ 原因：大铜箔吸热多，峰值温度容易不够导致虚焊

存入知识库，新人用自然语言提问，系统检索并回答。
