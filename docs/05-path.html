05 · 路径：技术演进 5 步走

不要一步到位全自动，按成熟度逐级演进，本项目试点只做到「阶段3」。

1

数据化

系统只记录：设定值 + 实测曲线 + 质量结果。人照常调炉。信任级别：无（纯记录）。

2

建模

离线预测：这条曲线会不会出缺陷。人看报告。信任级别：低（参考）。

3

辅助推荐 ← 本项目做到这

给新板推荐参数 + 风险提示，人确认/微调后下发。信任级别：中（人把关）。

4

闭环自动

自动下发 + 实时微调补偿设备漂移。人监督、异常处理。信任级别：高（系统主导）。

5

数字孪生

上线前仿真验证曲线再下发。信任级别：最高。

5.1 各阶段对照表
