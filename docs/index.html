🧭 导读（30 秒看懂）

回流焊炉有 7~10 个温区，老师傅要调每个温区温度、链速、风量，调出的"炉温曲线"决定焊点好坏。目前靠人，难招、波动大、换线慢、经验随人走。本项目用数据 + AI 把这套能力做成系统：记录数据 → 建模预测缺陷 → 给新板推荐参数 → 人确认下发 → 持续迭代。先从"辅助推荐"做起，不一步到位全自动。

📊 全文含 16 张示意图（炉温曲线、四层架构、数据追溯、甘特时间轴、演进路径、反馈闭环、数据飞轮、经验萃取流程、部署拓扑、CI/CD 工作流，及 6 张系统 UI 原型：辅助推荐 / 数据源接入 / 模型训练评估 / 实时监控看板 / 知识库问答 / 质量反馈分析）。系统不只复制老师傅，更在稳定、全局寻优、可解释、自进化上超越——看不懂文字，就先看图。技术细节见 技术方案 V2（含完整 SQL DDL 与 API Schema）。
