1.3 采集质量红线（8 条自动校验）

以下任一条不满足，采集网关须标记该炉次为异常并告警，不进入训练集；人工复核后可放行或剔除。

  1. run_id 唯一且必填，格式 R-YYMMDD-NNNN，重复即冲突。
  2. 曲线完整性：每通道采样点 ≥ 60 点，否则判采集失败触发重测。
  3. 物理合理性：实测峰值与设定峰值偏差 > 30℃ 报警（疑似传感器/炉子异常）。
  4. 追溯率 100%：质量结果 lot_id 必须能关联到 run_id。
  5. 时间对齐：炉子/Profiler/服务器统一 NTP，时间戳误差 < 1s。
  6. 缺失通道补值：按同炉次邻近通道均值补，并置 imputed=1 标记。
  7. BGA 板双检：AOI 与 X-Ray 对 BGA 板至少一项须存在。
  8. BOM 完整性：板厚/铜面积/BGA 数缺失 → 质量模型降级为"仅曲线特征"模式。

第二部分 · AI 算法处理逻辑清单

模型层 · 三个核心模型 + 端到端链路

算法口径与 V2 §5~§7 一致。每个模型给出输入 → 处理逻辑 → 输出 → 约束 → 验收，可直接作为算法开发与 Code Review 的核对基准。

2.1 端到端处理链路

采集 → 落库(MySQL+InfluxDB) → 特征计算(profile_feature) → 模型推理(代理→质量→推荐) → 人确认下发(audit) → 质量反馈回灌(数据飞轮)

反馈闭环：每次下发后的 AOI/X-Ray/ICT 结果回流到 quality_result → 重训练样本 → 模型月月更准（见 10 改进）。

2.2 模型1 · 工艺代理模型（Surrogate）

项内容
目标设定值（温区×10 + 链速 + 风量）→ 预测 6 通道炉温曲线
输入11 维设定 +（可选）环境温/湿；维度与 reflow_run 一致
处理逻辑① 物理热模型 SimpleThermalSolver 出基线（传热方程，保物理下限）
② 神经网络残差 residual_nn 校正（PINN，6×180 输出）
③ 最终 = 物理基线 + NN 残差
输出6×180 预测曲线（供特征计算与推荐引擎评估）
训练数据历史所有炉次 (设定值, 实测曲线) 对；小样本阶段物理模型兜底，数据增多 NN 渐主导
约束 / 安全物理合理性：升温段须递增、峰值落合理区间；避免出现物理不可能的曲线（如峰值 400℃）
验收预测曲线 vs 实测 RMSE < 阈值（例：±5℃ 内占比 ≥ 95%）

2.3 模型2 · 质量预测模型（Quality）

项内容
目标曲线特征(8) + 板/锡膏元信息(8) → 各缺陷类型概率 + 预测良率
输入16 维（见 V2 §5.2）：peak_temp / TAL / ramp_up / ramp_down / delta_t / soak_temp / time_above_183 / curve_duration + 板厚 / 铜面积 / BGA 数 / 最大 BGA / 锡膏编码 / 元件密度 / 环境温度 / 湿度
处理逻辑多标签 XGBoost（MultiOutputClassifier），每缺陷一个二分类；scale_pos_weight=15 加权补偿坏板稀少（坏:好≈1:30）；early_stopping_rounds=30
输出虚焊 / 桥连 / 立碑 / 空洞 / 锡珠 五类概率 + 预测良率；附 SHAP 归因（可解释）
训练数据3.2 万+ 炉次（曲线特征 + 质量标签）；坏板过采样/合成补偿
约束 / 安全样本不平衡处理；输出概率须可审计回溯；BOM 缺失时降级为"仅曲线特征"
验收每缺陷类型 AUC ≥ 0.85，召回率 ≥ 0.80

2.4 模型3 · 参数推荐引擎（Recommender）

项内容
目标新板 BOM + 锡膏 + 工艺窗口 → top-k 最优温区设定 + 链速
输入product_id, solder_paste, bom{...}, constraints{peak_min/max, tal_min/max, ...}（见 V2 §9 recommend 请求）
处理逻辑贝叶斯优化在代理模型上搜 11 维空间：
① 代理模型预测曲线 → ② 提取特征 → ③ 质量模型预测缺陷概率 → ④ 工艺窗口罚分 → 目标=最小化缺陷概率
init_points=12, n_iter=36（≈10⁶ 等效评估，毫秒级/次）
输出top-3 设定 + 预测良率 score + SHAP 解释（top3 特征）+ 风险清单（类型/概率/缓解）
约束 / 安全硬约束注入：peak∈[235,250], TAL∈[45,90], ramp_up∈[1,3]；越界 100% 拦截不进入结果；异常自动回退历史最优设定
验收推荐炉次平均良率 ≥ 同产品老师傅手调 +1.5pp；良率波动 σ 降低 ≥40%
RAG 知识库（问答侧，非工艺模型）：本地 Qwen-7B 大模型 + BGE-M3 Embedding（1024d）→ pgvector 检索 Top-5 → 带引用回答。全链路本地，数据不出厂。若贵司已切换其他大模型，请同步更新此处选型（见文末说明）。
