11 · 术语表（小白也能看懂）

把文档里出现的黑话全部翻译成大白话，分 SMT 工艺、AI/数据、项目三类。

SMT / 回流焊
术语大白话解释
SMT表面贴装技术，把元件贴到电路板上的工艺总称。
回流焊（Reflow）用炉子把焊锡膏加热熔化、冷却后把元件焊牢的工序。
炉温曲线（Thermal Profile）过炉时板子温度随时间变化的曲线，决定焊得好不好。本系统核心对象。
温区（Zone）回流炉分几段加热区，常见 7~10 个，每段可单独设温度。
链速（Conveyor Speed）传送带速度，决定板子在炉里待多久，直接影响升温时间。
锡膏（Solder Paste）焊锡粉+助焊剂组成的膏体，印刷在焊盘上，受热熔化形成焊点。
无铅锡膏 SAC305常用无铅合金，熔点约 217℃，比有铅更怕温度不准。
峰值温度（Peak Temp）曲线最高点温度，太低虚焊、太高损坏元件。
TAL温度高于液相线（217℃）的总时间，太短焊不牢、太长元件受损。
升温/降温斜率每秒升/降几度，太快易立碑、太慢助焊剂失效。
ΔT（板面温差）同一块板上不同位置的最大温差，太大说明受热不均。
BGA / QFN底部焊球/无引脚的芯片，看不见焊点，靠 X-Ray 查，最怕空洞、虚焊。
空洞（Void）焊点里气泡，BGA 常见，太多影响散热和可靠性。
虚焊 / 桥连 / 立碑三种典型缺陷：没焊上 / 相邻引脚连在一起 / 元件一头翘起。
SPI / AOI / X-Ray三种检测设备：查锡膏印刷 / 查外观 / 查内部焊点。
炉温测试仪（Profiler）带热电偶的小记录仪，随板过炉，测出真实炉温曲线。
工艺窗口（Process Window）温度/时间等参数的"安全合格范围"，超出就易出缺陷。
AI / 数据
术语大白话解释
特征（Feature）喂给模型的数据指标，如峰值温度、TAL 等。
标签（Label）已知结果，如这块板"合格/虚焊"，用来教模型。
训练 / 预测用历史数据教模型（训练）；用新数据让模型猜结果（预测）。
代理模型（Surrogate）用"设定值"快速算出"大概曲线"的简化模型，省去真过炉。
XGBoost / 随机森林两种常用机器学习算法，小样本也好用、能解释。
贝叶斯优化一种"聪明试参"方法，用最少次数找到最优参数。
AUC / 召回率衡量模型准不准的指标：AUC 越高越会分好坏，召回率越高越不漏坏板。
小样本坏板数据太少，模型难学，需用仿真/迁移学习补充。
迁移学习把别的产品/炉子学来的经验，挪用到本场景。
RAG（检索增强）让大模型先查"老师傅经验库"再回答，减少胡说。
数字孪生（Digital Twin）在电脑里建一个虚拟炉子，先仿真再真干。
闭环控制系统自己根据实测微调参数，不需人每次干预。
项目 / 管理
术语大白话解释
WBS工作分解结构，把大项目拆成可干活的小任务清单。
RACI谁负责(R)、谁批准(A)、咨询谁(C)、知会谁(I) 的分工矩阵。
试点（Pilot）先在小范围试跑，验证可行再推广，降低风险。
基线（Baseline）改进前的原始水平，用来对比"有没有变好"。
SMART目标要具体、可衡量、可达成、相关、有时限。
ROI投资回报率，省的钱 ÷ 花的钱。
