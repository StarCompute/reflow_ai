01 · 缘起：我们为什么要做这件事

用大白话讲清问题，让不懂技术的人也能秒懂痛点。

1.1 现状痛点

回流焊炉有 7~10 个温区，老师傅要调：每个温区设多少度、传送带多快（链速）、风量多大。调出来的"炉温曲线"决定焊点好不好。目前靠人，问题有四：

痛点具体表现量化影响
难招难留培养一名能独立调炉的师傅需 2~3 年；月薪普遍 1.5万~2.5万一人离职，整线质量下滑 1~2 周
波动大不同人、不同班次调出的曲线不一致同产品返修率波动可达 ±1.5%
换线慢新板/新锡膏上线，老师傅反复试 3~5 炉才稳定每次换线浪费 2~4 小时 + 报废样板
经验随人走老师傅的"小诀窍"在脑子里，没记录知识无法复制，新人从头学

1.1.1 老师傅调出的"炉温曲线"长什么样

上面说的"曲线"就是下面这条线——横轴是时间、纵轴是板子温度。老师傅调温区和链速，本质就是在"雕刻"这条线的形状。这条线必须落在工艺窗口里（见右侧红线、黄线标注），否则就会虚焊、桥连、立碑。

1.2 行业背景

为什么现在更要做

① 无铅锡膏（SAC305，熔点 217℃）对温度更敏感，工艺窗口比有铅更窄。

② 元件越来越复杂（BGA、QFN、01005 小元件），虚焊、空洞、立碑风险更高。

③ 工厂招工难、人力成本涨，用系统替代重复经验决策是必然趋势。

1.3 一句话缘起

老师傅的能力 = 「看板子 → 想曲线 → 调参数 → 预判风险」。
这套能力可以被 数据化 + 建模，做成系统，让机器辅助甚至替代决策。
